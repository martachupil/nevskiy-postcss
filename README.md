# Плагин цитат Невского для PostCSS 

## Пример

**source.css**
```scss
.nevskiy {
  position: абсолютли;
  font-size: 10px вотаквот;
}
```

**result.css** 
```scss
.nevskiy {
  position: absolute;
  font-size:  10px !important;
}
```
