# Плагин цитат Невского для PostCSS 

## Пример
**source.css**
```
.nevskiy {
  position: абсолютли;
  font-family: 10px вотаквот;
}
```
**result.css** 
```
.nevskiy {
  position: absolute;
  font-family:  10px !important;
}
```
